<h2>🛠️ Funcionalidades do Projeto</h2>

- 🎮 <code>Modo de Jogo</code>: Dois jogadores humanos alternam turnos para marcar X ou O no tabuleiro.
- 🧠 <code>Lógica de Jogo</code>: Verificação de condições de vitória, empate e alternância de turnos.
- 🖥️ <code>Interface de Console</code>: Interação simples via terminal para inserção de jogadas e exibição do tabuleiro.
